Rent A Car

Write a program that displays the following menu for a parking area:

  1. M = Motorcycle
  2. C = Car
  3. B = Bus

The program inputs the type of vehicle and number of days to park the vehicle. It finally displays the total chares for the parking accorkding to the following:

  1. Motorcycle Rs.10 per day
  2. Car Rs. 20 per day
  3. Bus Rs. 30 per day
